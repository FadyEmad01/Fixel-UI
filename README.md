## ✨ About

**Fixel UI** is a modern UI component distribution platform inspired by the best developer-first libraries.

It gives you access to beautifully crafted, fully animated components that you can:
- Copy
- Customize
- Use instantly in your projects

No heavy dependencies. No lock-in. Just clean, production-ready UI.

---

## 🚀 Features

- 🎬 Smooth animations powered by Motion
- 🎨 Tailwind CSS for styling
- 🧠 TypeScript-first architecture
- 🧩 Shadcn CLI integration
- 📦 Copy & paste components
- ⚡ Optimized for performance
- 🌍 Open-source
